# Artifact Packages

This directory is reserved for bounded artifact packages that are developed inside the DSR Framework repository before they are mature enough to warrant a separate repository.

Artifact packages in this directory are distinct from `examples/`. Examples demonstrate use of the framework. Artifact packages may contain independently reviewable theory, model, validation, or domain artifacts that use the framework and may later be split into standalone repositories when they have their own audience, release cadence, license posture, DOI, evaluation record, or platform package.

The presence of an artifact package here does not expand the DSR Framework v1.x package claim. The root package remains the released DSR Framework artifact package, currently claiming L4 reusable-stable conformance and not claiming L5 archival/publication-ready status.

Each artifact package should include its own README, artifact profile, local inventory, retained records, source basis, validation evidence, and explicit non-claims.
